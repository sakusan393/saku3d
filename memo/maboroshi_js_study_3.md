---

8bitカラー(256色)の近似値をとるような処理
(内訳 R:2bit G:3bit B:3bit)

```
// 色数を減らす処理
float ratioR = 4.0;
float ratioG = 8.0;
float ratioB = 8.0;
destColor.r = float(int(destColor.r * ratioR)) / ratioR;
destColor.g = float(int(destColor.g * ratioG)) / ratioG;
destColor.b = float(int(destColor.b * ratioB)) / ratioB;
```
